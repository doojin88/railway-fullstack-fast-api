# FastAPI Full-Stack Template for Railway

A production-ready full-stack template based on the [FastAPI Full-Stack Template](https://github.com/fastapi/full-stack-fastapi-template) optimized for deployment on Railway.

## Features

- **Backend**: FastAPI with SQLModel, PostgreSQL, and Pydantic
- **Frontend**: React with Vite and TypeScript
- **Always Up-to-Date**: Dockerfiles clone the latest template on each build
- **Railway Optimized**: No cache mounts for seamless deployment

## Quick Deploy

[![Deploy on Railway](https://railway.app/button.svg)](https://railway.app/new/template)
